Dataset · ข้อมูลที่ใช้
Dataset: synthetic_netflix_like_thai_reviews_5000
ภาษา: ภาษาไทย (Thai)
ประเภท: รีวิว Netflix (Binary Sentiment: Positive / Negative)
จำนวนข้อมูล: 5,000 รายการ (สมดุล 50/50)
Preprocessing: Whitespace normalization + Repeated punctuation reduction
Vectorizer: TF-IDF Word (1-2 gram) + Char (2-4 gram) รวมกัน
Model: Logistic Regression (C=1.0, class_weight=balanced)
